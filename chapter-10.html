第10章 为文本添加样式

选择字体系列

步骤：font-family:name;
对于包含多个单词的字体名称，应该用引号（单引号或双引号）包围起来
虽然font-family属性是继承的，但有几个元素不会继承父元素的字体设置，其中有表单的select、textarea和input元素
不过可以强制它们继承父元素的字体设置，代码为input, select, textarea { font-family:inherit; }

指定替代字体

指定的字体不一定在每个系统上都能显示出来，因此，可以在font-family属性中列举一种以上的字体，浏览器会使用列表中第一个已安装在访问者系统内的字体或Web字体
字体列表成为字体栈，通常字体栈至少包含三个字体
步骤：font-family:name1, name2, name3;，用一个逗号和空格分隔每个字体，最后以一个表示类属的标准字体(serif、sans-serif、cursive、fantasy或monospace，它们代表首选字体最为接近的风格)结束字体列表，其中cursive、fantasy、monospace分别表示手写字体、装饰字体和等宽字体
几乎可以确定Arial是万维网上用的最多的字体，因为font-family:arial, helvetica, sans-serif;这条声明可谓随处可见
但设计师通常更愿意使用Helvetica，但Windows上一般没有这个字体，所以使用font-family:sans-serif会让所以操作系统使用无衬线字体——Windows显示Arial，而OSX显示Helvetica

创建斜体

步骤：font-style:italic;或font-style:oblique;，99%的情况下都会使用italic，在所以情况，很难察觉使用oblique和italic的差异
font-style是继承的，如果想取消从父元素中继承了的斜体，可以使用font-style:normal;

应用粗体格式

步骤：font-weight:bold;，也可以输入100~900之间的100的倍数，其中400代表正常粗细，700代表粗体，或者输入bolder（更粗）或lighter（更细）以设置相对当前粗细的值
font-weight是继承的，如果想取消，可以使用font-weight:normal;，有些Web字体服务，如Google Fonts，要求使用font-weight:400;

设置字体大小

为网页里的文本设置字体大小有两种方法：(1)可直接使用像素指定要使用的特定字号(2)使用百分数或者em或者rem指定相对于父元素文本的大小
建议使用em这样的相对单位，相对单位可以有更大的灵活性，有助于建立在各种设备都能显示良好的页面
使用em或百分数这种方法时，最好在body元素上建立一个基准，即声明body { font-size:100%; }，大多时候，这项设置等价于将字体大小设为16px，即大多数系统的默认字体大小
font-size是继承的，该属性值传递给其他元素，除非其他元素通过浏览器默认设置了自己的font-size，或在样式表指定了自动的font-size
1em=16px，em值=要指定的字体大小/父元素的字体大小，如果设置段落p的字体大小为17px，则该段落内1em=17px
大多数情况下，100%的body font-size等于16px，不过有一种例外的情况，即用户对浏览器的设置覆盖了该默认值，例如，如果用户是视障人士，他们可能将默认字体大小设置为22px
rem(root em的简称)，它同em很像，不过它总是以根元素为参照系设置其他元素的字体大小，而不是父元素，根元素是html元素，因为它包含了body，也就包含了所有内容，这样就简化了字体大小的设置，因为html字体大小通常不会变，不像父元素的大小是不确定的，设置根元素的字体大小：html { font-size:100%; }，虽然现代浏览器对它的支持程度很高，但Internet Explorer知道IE9才开始支持，因此，可以针对旧版本的IE提供以px单位的值，再紧跟着为现代浏览器准备的rem值，例如：p { font-size:17px; font-size:1.0625rem;}
rem值=要指定的字体大小/根元素字体大小
指定特定字体大小：font-size:npx;，或使用关键字指定字体大小，即使用xx-small、x-small、small、medium、large、x-large或xx-large
根据父元素设置字体大小：font-size:n%;或font-size:nem;
根据根元素设置字体大小：font-size:nrem
磅(pt)用做打印样式表的单位

设置行高

行高指的是段落的行距，即段落内每行之间的距离
步骤：line-height:n;，这里的n是一个数字，它与元素的字体大小相乘，得出需要的行高
或者输入a，这里的a是以em、像素或磅为单位的值
或者输入p%，这里的p%是字体大小的百分数
通常是使用数字设定行高，所有的子元素都会继承这个因子，如果父元素字体大小是16px行高1.5，则行高=16×1.5=24px，子元素的字体大小如果是10px，则行高=10*1.5=15px
如果使用百分数或em值，那么只会继承产生的行高，如果父元素的字体大小事16px，行高150%，相当于16×150%=24px，那么子元素也是24px

同时设置所有字体值

可以使用font简写属性同时设置字体样式、粗细、变体、大小、行高和字体系列，font属性是继承的
使用font简写属性不要求指定所有字体属性，但至少应该包含字体大小和字体系列属性，先是字体大小，再是字体系列
步骤：font:font-size(/line-height) font-family (font-weight font-style font-variant)，其中括号里的是可选的
font-variant表示设置小型大写字母，可选的值有normal(默认值)和small-caps(显示小型大写字母)
设置时行高是可选的，但它如果出现，就必须紧跟在字体大小和斜杠后面，比如：font:1.25em/2.5 "Palatino Linotype", Palatino, serif;

设置颜色

可以修改网页中文本的颜色，样式表可以包含颜色名称、十六进制值、RGB、HSL、RGBA和HSLA值的任意组合来定义颜色
步骤：color:colorname;，这里的colorname可参考第7章中的CSS颜色
除了link元素(a:link)，其他元素都会继承body元素中的color属性，开发人员必须显示设置link元素的color属性

设置背景

1.修改文本的背景颜色：background-color:transparent或color;，这里transparent表示允许透过父元素的背景颜色，color表示颜色值
如果您不希望某元素拥有背景色，同时又不希望用户对浏览器的颜色设置影响到您的设计，那么设置 transparent 值还是有必要的。
2.使用背景图像作为背景：background-image:url(image.png);，这里的image.png是图像相对于样式表所在位置的路径和文件名，或输入none表示不使用图像
背景图像默认是重复的
3.重复背景图像：background-repeat:direction，这里的direction可以取四个值：repeat、repeat-x、repeat-y和no-repeat，分别表示同时横向和纵向重复图像、只横向重复图像、只纵向重复图像、不重复图像
4.控制背景图像是否随页面滚动：background-attachment:fixed或scroll或local，fix表示背景图像会附着在浏览器窗口上，也就是说即使访问者滚动页面，图像任然会继续显示，scroll表示访问者滚动页面时背景图像会移动，local表示只有访问者滚动背景图像所在元素（而不是整个页面）时，背景图像才移动，默认值是scroll
5.指定元素背景图像的位置：background-position:x y;，其中x和y可以表示为距离左上角的绝对距离或百分数，也可以用left、center、right表示x，用top、center、bottom表示y
可以在background-position属性中使用负数，例如background-position:-45px 80px;将图片定位在元素左边外侧45像素、顶部内侧80像素
6.指定元素背景图像的显示尺寸：background-size:value;，属性值可以是：
contain：在显示图像完整宽度和高度的情况下，尽可能地扩大图像的尺寸，背景图像可能不会填充整个背景区域
cover：在填充元素整个背景区域的情况下，让图像尽可能地小，图像的一部分可能会超过元素的范围而不可见
length：设置背景图像的高度和宽度，第一个值设置宽度，第二个值设置高度，如果只设置一个值，则第二个值会被设置为"auto"
percentage：以父元素的百分比来设置背景图像的宽度和高度，第一个值设置宽度，第二个值设置高度，如果只设置一个值，则第二个值会被设置为"auto"
7.background-clip和background-origin两个属性分别控制元素背景显示的范围和开始的位置，这两个属性都接受相同的值：content-box包含内容、padding-box包含内容和内边距、border-box包含内容、内边距和边框，background-clip的默认值为border-box，而background-origin的默认值为padding-box，详情请查看W3School的说明
8.在一条声明中设置所有背景属性：background:指定任何可接受的background属性值
background-color默认transparent(透明)、background-image默认none(无)、background-repeat默认repeat(重复)、background-attachment默认scroll(滚动)、background-position默认0 0(等价于top left，即左上角)、background-size默认100% 100%(原图大小)
建议使用background简记法，不必指定所有的属性，没有指定的属性都将设置为默认值

控制间距

可以增加或减少单词之间或字母之间的距离，前者称为字间距，后者称为字偶距
1.指定字间距：word-spacing:length;这里的length是一个带单位的数字，如0.4em或5px，例：It is a sentence
2.指定字偶距：letter-spacing:length;这里的length是一个带单位的数字，如0.4em或0.5px，例：It is a sentence
可以对单词间距和字母间距使用负数，间距可能会受到所选对齐方式和字体系列的影响
要将字母间距或单词间距设为默认值，可以使用normal或0
word-spacing和letter-spacing属性是继承的，如果使用em值，那么只有结果大小(即计算出来的值)会被继承

添加缩进

通过设置text-indent属性，可以指定段落第一行前面应该空出多大的间距
步骤：text-indent:length;这里的length是一个带单位的数字，如1.5em或18px，要忽略缩进，使用0即可
我们可以为其他元素应用text-indent属性，而不仅是段落元素，但默认情况下对em、strong、cite等内联元素没有效果，可以将它们设置为display:block;或display:inline-block;就可以强制为其应用text-indent属性 使用负数会产生悬挂缩进，使用悬挂缩进时，可能还需要增加文字框周围的内边距或外边距，从而让容器可容纳伸到外边的文本
与通常情况一样，em值要根据元素的字体大小进行计算，百分数要根据父元素的宽度进行计算
text-indent属性是继承的，如果要使用百分数或em值，那么只有生成的大小(即计算出来的值)会被继承

对齐文本

根据需要，可以让文本左对齐、右对齐、居中对齐或两端对齐
步骤：text-align:value;，其中value可以是：left表示左对齐、right表示右对齐、center表示居中对齐、justify表示两端对齐，text-align属性是继承的
如果选择让文本两端对齐，要注意单词间距和字母间距有可能受到严重的影响
text-align对适用于盒模型的元素（如：h1~h6、p等，显示为单独的行）来说是起作用的，但对短语内容元素（如span、strong、em、cite等，显示在行内）来说是不起作用的，可以将它们的样式改为display:blcok;（显示单独的行）或display:inline-block;（这个要设置一定的宽度）

修改文本的大小写

使用text-transform属性，可以为样式定义文本的大小写，可以将文本显示为首字母大写、全部大写、全部小写或按原样显示
步骤：text-transform:value;这里value可以是：capitalize让每个单词的首字母大写、uppercase让所有字母大写、lowercase让所有的字母小写、none让文本保持本来的样子
text-transform属性是继承的，可以将属性值设为none来取消继承的值

使用小型大写字母

使用font-variant可以调用小型大写字母变体，使一些大写的字母缩小到小写字母的大小,比如It Is A Sentence.
步骤：font-variant:small-caps;，这个属性是继承的，如果要取消，可以令属性值为none

装饰文本

可以使用text-decoration属性装饰文本，如添加下划线或其他类型的线条
步骤：text-decoration:value;这里value可以是：underline下划线、overline上划线、line-through删除线
text-decoration属性是继承的，可以令属性值为none来取消文本装饰

设置空白属性

默认情况下，HTML文档里的多个空格和回车会显示为一个空格或者被忽略，如果要让浏览器显示这些额外的空额，可以使用white-space属性 步骤：white-space:value;这里value可以是：pre让浏览器显示原文本中的所有空格和回车、nowrap确保所有空格不断行，也就是文本全部显示在一行、normal按正常方式处理空格
